Setup instructions
  1. In Resend dashboard → Webhooks → Add Endpoint.
  2. URL: https://vgjmfpryobsuboukbemr.supabase.co/functions/v1/resend-webhook
  3. Events: email.sent, email.delivered, email.bounced, email.complained, email.opened, email.clicked, email.delivery_delayed
  4. Copy the signing secret into Supabase → Edge Functions → Secrets as RESEND_WEBHOOK_SECRET.
  5. Until the secret is set, events still land here but with a webhook_secret_not_configured warning in edge logs.
